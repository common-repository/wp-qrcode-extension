== Description ==
WP-QRcode を拡張するプラグイン。QRコードの表示機能をサイドバーウィジェットとして提供します。前提として WP-QRcode がインストール・有効化されている必要があります。

== Installation ==
「WP QRcode」の公式サイトより「WP QRcode」をダウンロードし、事前に「WP QRcode」をインストール・適用して下さい。
「WP QRcode EXTENSION」をダウンロードし、WordPressにインストール・適用して下さい。
管理画面の [外観] - [ウィジェット] を開き、QRコードビューアをサイドバーなどにドラッグ＆ドロップして下さい。
